# AppEmisoras Cúcuta

## 1. Descripción general del proyecto

AppEmisoras Cúcuta es una aplicación móvil diseñada para centralizar y facilitar el acceso a las distintas emisoras de radio de los colegios de la ciudad de Cúcuta, Colombia. El objetivo principal es crear un espacio donde la comunidad educativa (estudiantes, padres, profesores y directivos) pueda conectarse, informarse y disfrutar de los contenidos producidos por las instituciones educativas de la ciudad.

El proyecto busca fortalecer la comunicación y la identidad de cada colegio, promoviendo la participación estudiantil en la creación de contenidos radiofónicos y ofreciendo una plataforma moderna y accesible para su difusión.

## 2. Tecnologías usadas

Para el desarrollo de esta aplicación se utilizarán las siguientes tecnologías:

*   **Lenguaje de programación:** Kotlin y Java
*   **Framework:** Android Nativo
*   **Base de datos:** Firebase Firestore
*   **Almacenamiento en la nube:** Firebase Cloud Storage
